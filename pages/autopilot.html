Autopilot Quick Reference

Complete reference for all six autopilot modes — AP, HDG, NAV, ALT, APR, and ATH. Panel located on the left side of the main instrument panel (item 1 in cockpit layout). Four selector knobs for CRS, HDG, ALT, and SPD on lower center panel (item 5).

Selector knobs (lower center panel, item 5): CRS · HDG · ALT · SPD. Rotate each knob to set the target value before engaging the corresponding mode.
AP
AP — Autopilot Master
  • Toggles the entire autopilot system on or off
  • Must be ON before any other mode will function
  • Green emissive indicator on the button when active
  • PFD top strip shows AP annunciation when engaged
  • Disengage before touchdown — press the button or use the disconnect switch on the stick
AP ALT HDG NAV ATH APR ↑ AP annunciation appears here when master is on
HDG
HDG HOLD — Heading Hold
  • Holds the aircraft on the heading set by the HDG selector knob
  • Rotate HDG knob to change the target heading — aircraft follows
  • Use for: ATC vectors, departure headings, pattern work, missed approach headings
  • Automatically drops out when the ILS localizer is captured in APR mode
  • Do not re-engage HDG after LOC capture on ILS approach
AP ALT HDG ATH HDG [257] ← target heading
NAV
NAV HOLD — Navigation Hold
  • Tracks the course deviation needle shown on the PFD compass rose
  • In NAV1 mode: tracks the VOR radial set in FMS NAV page CRS field
  • In GPS mode: tracks the FMS flight plan GPS deviation path
  • Check the PFD nav mode softkey first — it determines what NAV HOLD follows
  • Use for: RNAV approaches, VOR approaches, en route navigation
  • Never use APR as a substitute for NAV HOLD on non-ILS approaches
AP ALT NAV ATH GPS TRACK or VOR1 CRS 359
Confirm PFD softkey matches your intended source before engaging NAV HOLD. Wrong softkey = wrong tracking.
ALT
ALT HOLD — Altitude Hold
  • Holds the altitude the aircraft is at the moment you press the button
  • Pre-select target on ALT selector knob — aircraft will level at that altitude
  • PFD shows ALT in the altitude mode annunciation when active
  • Changes to G/S automatically when ILS glideslope is captured in APR mode
  • For RNAV / VOR step-downs: disengage ALT to descend, re-engage at each step altitude
AP ALT HDG ATH ← normal AP G/S APR ATH ← ILS glideslope captured
APR
APR — Approach Mode (ILS ONLY)
  • Arms and then captures the ILS localizer and glideslope
  • APR box on PFD: white = off, orange = LOC armed, green = LOC captured
  • When LOC is captured: HDG HOLD drops out automatically
  • When G/S is captured: ALT HOLD changes to G/S automatically
  • Requires ILS frequency tuned in FMS NAV page VOR1
  • Requires PFD nav mode softkey set to ILS
APR armed: box = ORANGE APR active: box = GREEN G/S captured: ALT changes to G/S
APR mode only works with an ILS signal. For RNAV or VOR approaches use NAV HOLD. Pressing APR for non-ILS approaches wastes time and provides no tracking.
ATH
ATH — Autothrottle
  • Holds the airspeed the aircraft is flying at the moment of engagement
  • Rotate SPD selector knob to change the target airspeed
  • Critical limitation: only operative above 2,500 ft AGL
  • Disengages automatically when descending through 2,500 ft AGL
  • Set VREF on the SPD knob before final — manage throttle manually on short final
  • Use A/T for: en route cruise, descents, approach setup above 2,500 ft
A/T disengages below 2,500 ft AGL. If you rely on it to maintain VREF on short final and forget it has cut off, you will lose speed rapidly. Monitor airspeed continuously on final approach.
