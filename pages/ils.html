ILS Approach Configuration Guide

Step-by-step FMS, NAV page, and autopilot setup for a full ILS approach from vectors to touchdown or go-around. Primary example: KDYS ILS RWY 34L (110.3 / I-DYS / CRS 359°).

Phase 1 — Pre-approach setup
1
FMS APPR PAGE — flaps, VREF, DH, QNH
FMS — PERF → CRZ PG → APPR PG
APPR PAGE FLAPS --> 17° VREF --> □□□ Kt ← click to auto-compute DH □□□□ RADIO ← type DH (e.g. 200) LAND QNH --> 29.92 ← from ATIS <--CRZ PG
  • Press PERF key on FMS CDU
  • Navigate right → CRZ PG → APPR PG
  • Left key to toggle flaps → 17°
  • Click VREF — auto-computes for weight + flap config
  • Type DH from approach plate (e.g. 200)
  • Update QNH from current ATIS
2
FMS NAV PAGE — tune ILS frequency and course
FMS — press NAV key
RADIO NAV VOR1/FREQ CRS [I-DYS]/110.3 359 VOR2/FREQ CRS [ ]/[ ] --- ADF1/FREQ BRG [ ]/890 ---
  • Press NAV key on FMS CDU
  • Type ILS freq: 110.3 → left key → VOR1
  • Confirm identifier shows I-DYS
  • Type inbound course 359 → right key → CRS
  • Verify CRS needle on PFD compass rose
3
PFD — switch nav mode to ILS (lower softkey)
PFD lower softkeys
VSD | ADF | [ILS] | NAV1 | GPS ↑ press this softkey
APR box on PFD: white = unarmed, orange = LOC armed, green = LOC captured.

Phase 2 — FMS approach loading
1
FPLN → destination → STAR screen
KDYS STARS RWY SID TRANS --- ------ ------ AVAILABLE RUNWAYS ←34L 3048M ←16R 3048M <RETURN
  • Press FPLN → scroll to destination KDYS → left key
  • Select RWY 34L
  • Select STAR (or NONE for vectors)
  • Select approach: ILS 34L
  • Press INSERT right key to activate
2
Verify FPLN shows approach fixes
FROM KDYS RWY 34L ...en route... MUCHO (FAF) DIST 5NM RWY34L DIST 5NM ----END OF ROUTE----
  • Confirm MUCHO (FAF) in flight plan
  • Confirm RWY34L as final fix
  • PFD nav mode must be set to ILS not GPS
If PFD shows GPS instead of ILS — APR mode will not capture the localizer. Switch to ILS softkey.

Phase 4 — Localizer intercept
1
Set HDG bug to ATC intercept heading
  • Rotate HDG selector knob (lower center panel) to ATC heading
  • Confirm HDG bug on PFD compass rose
  • HDG HOLD must be active (green)
  • Intercept angle: ≤ 30° to inbound CRS 359°
  • Speed: set A/T to VREF + 20 KIAS
Ideal intercept: 15–30° to localizer from base leg.
2
Monitor LOC needle — capture criteria
  • LOC needle appears within ~10 NM of transmitter
  • APR armed (orange) before needle moves
  • At ½ dot: AP initiates capture turn
  • Rollout within 30° of CRS 359°
If needle reaches full-scale before capture: disconnect AP and hand-fly the intercept.
Phase 5 — Final approach to DH
1
Mandatory altitude callouts — FAF to DH
  • At FAF (MUCHO) — "Gear down, Flaps 17, Cleared ILS 34L"
  • 1,000 ft AGL — "One thousand feet"
  • 500 ft AGL — "Five hundred, stable" or "Go-around"
  • DH (200 ft HAT) — "Minimums" then visual call or go-around
Any stable approach criterion not met by 500 ft AGL = mandatory go-around.
2
Instructor monitoring — common errors
  • Wrong NAV mode — "Check PFD softkey — you need ILS not GPS"
  • APR not armed — "Press APR on autopilot panel"
  • Low on G/S — "You're low on glideslope — check needle"
  • DH bust — "Go around — runway not in sight"
