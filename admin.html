📋 How to apply changes to GitHub
Most settings here are saved in your browser's local storage so they appear when you visit the site from this computer. To make them permanent for everyone:

1. Use the Card Builder to generate HTML → copy → paste into GitHub
2. Use the Colors section to generate a CSS snippet → paste into assets/style.css on GitHub
3. Use the Navigation section to generate nav HTML → paste into all page files on GitHub
4. Slideshow images and logos save in browser storage automatically when you click Save All
