FMS Setup Guide

Complete reference for every FMS page — from IDEN startup through FPLN route building, PERF V-speeds, APPR setup, COM/NAV tuning, and the DIR and PSN pages. The CDU is on the center pedestal (item 4 in cockpit layout).

IDEN page and INIT page — startup sequence
1
IDEN page — first thing you see on startup
Press IDEN shortcut key — or it shows automatically
BONE-1B ENGINES G-E F101 ACTIVE DATA BASE SEP050CT02/24 DB → INIT PAGE BLOCK D GNSS/JAMM3.4
  • Confirms aircraft type: BONE-1B
  • Engine type: G-E F101
  • Shows active navigation database date — verify it is current
  • Press L4 key (left key next to → INIT PAGE) to proceed
2
INIT page — set departure, destination, cruise FL
Press INIT shortcut key — or press L4 from IDEN
ROUTE INIT FROM TO KDYS KTIK FLT IDENT B1 Jet FINAL FL FL PLN --> FL360
  • Type departure ICAO (e.g. KDYS) → press left key → goes into FROM box
  • Type destination ICAO (e.g. KTIK) → press right key → goes into TO box
  • Type cruise flight level (e.g. 360 for FL360) → left key → FINAL FL
  • Press right key or the FPLN shortcut key to continue to the flight plan page
FINAL FL = cruise altitude in hundreds of feet. FL360 = 36,000 ft. Enter 360, not 36000.
FPLN page — building and managing the route
1
Select SID from the origin airport
LNAV FOR KDYS N32°25.2 W099°51.1 ←SID NEXT WPT →→ AIRWAYS → ←FPLN PAGE
  • Press FPLN key to open the flight plan page
  • Press the left key next to the origin airport (KDYS)
  • Press ←SID left key to open SID selection
  • Select the runway (e.g. 34L) from the available runways list
  • Select the SID procedure name
  • Press INSERT right key to confirm and add to route
2
Add en route waypoints and airways
FROM KDYS RWY 34L KDYS34L DIST 11NM D196J 13 1127Z D196F 17 1132Z BAPAG 21 1135Z BPK 23 1137Z TO KTIK DIST 2980
  • Type waypoint ICAO ID using the keyboard → press the left key next to where you want to insert it
  • Use up/down arrow keys to scroll through the full flight plan
  • Press AIRWAYS → right key to add airway segments between fixes
  • Click the destination airport to add a STAR and approach
  • Press the right accept key to activate the full route
3
Load STAR and approach at destination
KDYS STARS RWY SID TRANS --- ------ ------ AVAILABLE RUNWAYS ←34L 3048M ←16R 3048M <RETURN
  • Press left key next to destination airport in FPLN
  • Select STAR from the list (or NONE for vectors-to-final)
  • Select the landing runway
  • Select approach type: ILS, RNAV, or VOR
  • Press INSERT right key to add to route
  • Verify approach fixes appear at the bottom of the FPLN
4
Activate the route
  • Once all fixes are loaded, press the right accept key to activate
  • The FMS FPLN page shows the route as the active flight plan
  • The MFD MAP view updates to show the route line
  • NAV HOLD / LNAV will now follow this route
If you edit the route after activation, you must re-activate with the accept key each time a change is made.
PERF pages — V-speeds, cruise, and approach data
1
PERF → TKOF PAGE — takeoff V-speeds
Press PERF key → shows takeoff page
TAKE OFF V1 --> □□□ FLAPS VR --> □□□ --> 0° V2 --> □□□ 17° DENS ALT AUTO -304 ft V SPEEDS --> CRZ PG-->
  • Right keys toggle flap setting → select 0° or 17°
  • Press V SPEEDS → right key for automatic computation based on current weight and flap config
  • Or type V1, VR, V2 manually into the left boxes
  • Density altitude shown for reference
  • Press CRZ PG → to continue to cruise page
2
PERF → CRZ PAGE — cruise altitude check
CRZ PAGE CRZ FL FL 360 CURRENT FL FL 002 <--TKOF PG APPR PG-->
  • Shows your planned cruise FL vs current FL
  • Verify before top of descent that cruise FL matches your clearance
  • Press APPR PG → right key before descent to set approach data
3
PERF → APPR PAGE — must fill before every approach
APPR PAGE FLAPS --> 17° VREF --> □□□ Kt ← click to auto-compute DH □□□□ RADIO ← DH or MDA from plate LAND QNH --> 29.92 inHg ← from ATIS <--CRZ PG
  • Toggle flaps to 17° with left key
  • Click the VREF field — auto-computes for current weight and flap config
  • Enter DH (for ILS) or MDA (for RNAV/VOR) from the approach plate
  • Enter QNH from current ATIS broadcast
  • Complete this page before you receive approach vectors
This is one of the most commonly forgotten pre-approach steps. Brief it in your descent checklist every time.
COM page and NAV page — radio management
COM
COM page — communication radios and transponder
Press COM shortcut key on CDU
RADIO COM ACTIVE STBY COM 1 COM 1 118.705 <-------> 124.850 COM 2 COM 2 124.850 <-------> 124.850 XPDR MODE XPDR CODE ALT 2000
  • Type frequency using dot notation (e.g. 118.10 or 119.3)
  • Left key → sets as ACTIVE immediately
  • Right key → sets as STANDBY
  • To swap active and standby: press CLR to empty scratchpad, then click the active frequency
  • Bottom left key cycles XPDR MODE (STBY / ON / ALT) — use ALT for normal flight
  • Type squawk code assigned by ATC → left key → XPDR CODE
NAV
NAV page — ILS, VOR, ADF, and TACAN tuning
Press NAV shortcut key on CDU
RADIO NAV VOR1/FREQ CRS [I-DYS]/110.3 359 VOR2/FREQ CRS [ ]/[ ] --- ADF1/FREQ BRG [ ]/890 --- TAC1/FREQ CRS [ ]/[ ] ---
  • Type VOR or ILS frequency → left key → assigns to VOR1 or VOR2
  • Station identifier auto-fills when the station is in range
  • Type OBS inbound course → right key → CRS
  • ADF: type frequency → left key → ADF1 (no course required)
  • TAC: TACAN for military approaches and aerial refueling rendezvous
For ILS: tune frequency in VOR1, set inbound course in CRS field. The same page is used for VOR — just a different frequency and identifier.
DIR page and PSN page
DIR
DIR page — fly direct to any waypoint instantly
DIRECT TO WAYPOINT →← FPLN WPTS →D196J →D196F →BAPAG →BPK →DATOB
  • Press the DIR shortcut key on the CDU
  • Use up/down arrows to scroll to the desired waypoint
  • Press the left key next to the waypoint you want
  • Press Direct Now at the bottom right key
  • FMS immediately draws a direct path — NAV HOLD follows it
Use DIR whenever ATC says "Cleared direct [fix]" or "Fly direct [waypoint]." It overrides the current route segment and goes straight to the selected fix, then resumes the flight plan from there.
PSN
PSN page — current GPS position
CURRENT POSITION 32.4648° -99.4841° --> CHANGE NOTATION --> STORE PRESENT POS NO POS° NO POS°
  • Shows the aircraft's real-time GPS coordinates
  • Press left key → changes coordinate notation format (decimal / degrees-minutes)
  • Press STORE PRESENT POS → saves current coordinates as a stored position
  • Stored positions can be used as waypoints in the flight plan or as weapon target coordinates
Useful for Phase 3 weapons events — fly over the target area, store the position on the PSN page, then enter those coordinates on the weapons MFD page.
MFD pages — right-hand display system pages
1
MFD softkey tabs — top of right-hand display
FLT ELEC FMS ENV ↑ ↑ ↑ ↑ Flight Electrical FMS Environment Surfaces System System (Pressurization)
  • Right-hand MFD softkeys at the top cycle through system pages
  • Left-side softkeys on the MFD cycle between: MAP/WX, TCAS, TFR, ENG, FLT, ELEC, FMS, ENV
  • Each page shows a different system status
2
