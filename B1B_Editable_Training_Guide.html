ILS Approach Configuration Guide
Step-by-step FMS, NAV page, and autopilot setup for a full ILS approach from vectors to touchdown or go-around. Primary: KDYS ILS RWY 34L (110.3 / I-DYS / CRS 359°).
Phase 1 — Pre-approach setup
1
FMS APPR PAGE — flaps, VREF, DH, QNH
FMS — PERF → CRZ PG → APPR PG
APPR PAGE FLAPS --> 17° VREF --> □□□ Kt ← click to auto-compute DH □□□□ RADIO ← type DH (e.g. 200) LAND QNH --> 29.92 ← from ATIS <--CRZ PG
  • Press PERF key on FMS CDU
  • Navigate right → CRZ PG → APPR PG
  • Left key to toggle flaps → 17°
  • Click VREF — auto-computes for weight + config
  • Type DH from approach plate (e.g. 200)
  • Update QNH from current ATIS
2
FMS NAV PAGE — tune ILS frequency and course
FMS — press NAV key
RADIO NAV VOR1/FREQ CRS [I-DYS]/110.3 359 VOR2/FREQ CRS [ ]/[ ] --- ADF1/FREQ BRG [ ]/890 ---
  • Press NAV key on FMS CDU
  • Type ILS freq: 110.3 → left key → VOR1
  • Confirm identifier shows I-DYS
  • Type inbound course 359 → right key → CRS
  • Verify CRS needle on PFD compass rose
3
PFD — switch nav mode to ILS (lower softkey)
PFD lower softkeys

APR box shows white when unarmed, orange when LOC is armed, green when captured.

Always verify approach plate before flight. Brief all fixes, DH, and missed approach before vectors.
Phase 2 — FMS approach loading
1
FPLN → destination → STAR screen
KDYS STARS RWY SID TRANS --- ------ ------ AVAILABLE RUNWAYS ←34L 3048M ←16R 3048M ←34R 1890M <RETURN
  • Press FPLN → scroll to destination KDYS → left key
  • Select RWY 34L
  • Select STAR (or NONE for vectors)
  • Select approach: ILS 34L
  • Press INSERT right key to activate
2
Verify FPLN shows approach fixes
FROM KDYS RWY 34L ...en route... MUCHO (FAF) DIST 5NM RWY34L DIST 5NM ----END OF ROUTE----
  • Confirm MUCHO (FAF) in flight plan
  • Confirm RWY34L as final fix
  • PFD nav mode must be set to ILS not GPS
  • FMS NAV page VOR1 confirms I-DYS / 110.3
If PFD nav mode shows GPS instead of ILS — APR mode will not capture the localizer. Switch to ILS mode on PFD softkey.

Phase 4 — Localizer intercept
1
Set HDG bug to ATC intercept heading
  • Rotate HDG selector knob (lower center panel) to ATC heading
  • Confirm HDG bug moves on PFD compass rose
  • HDG HOLD must be active (green)
  • Intercept angle: ≤ 30° to inbound CRS 359°
  • Speed: set A/T to VREF + 20 KIAS on vectors
Ideal intercept: 15–30° to localizer from base leg. A 30° cut at 10 NM gives clean capture without overshoot.
2
Monitor LOC needle — intercept criteria
▼ ← LOC needle moving ──────┼────── N VOR1 DME I-DYS 8.4 Full-scale = 2.5° off course ½ dot = 1.25° off course
  • LOC needle appears within ~10 NM of transmitter
  • APR armed (orange) before needle moves
  • At ½ dot: AP initiates capture turn
  • Rollout within 30° of CRS 359°
If needle reaches full-scale before capture: disconnect AP and hand-fly the intercept.
Phase 5 — Final approach to DH
1
Mandatory altitude callouts — FAF to DH
  • At FAF (MUCHO) — "Gear down, Flaps 17, Cleared ILS 34L"
  • 1,000 ft AGL — "One thousand feet"
  • 500 ft AGL — "Five hundred, stable" or "Go-around"
  • DH (200 ft HAT) — "Minimums" then visual call or go-around
Stable approach by 500 ft AGL
  • Localizer within ½ dot
  • Glideslope within ½ dot
  • Speed VREF ±5 KIAS
  • Config: gear down, flaps 17°
Any criterion not met by 500 ft AGL = mandatory go-around.
2
Instructor monitoring — common student errors
  • Low on G/S — chasing localizer, neglecting pitch
  • Speed decay — A/T off or set too low
  • APR not showing G/S — G/S not captured; still on ALT
  • Wrong NAV mode — GPS selected, no LOC needle
  • DH bust — no call, continuing below minimums
Correction phrases
  • "Check PFD softkey — you need ILS not GPS"
  • "APR not armed — press APR on autopilot panel"
  • "You're low on glideslope — check G/S needle"
  • "Go around — runway not in sight"
Non-Precision / APV Approach — KwikFlight B-1B
RNAV / GPS Approach Configuration Guide
FMS FPLN and NAV HOLD autopilot setup for GPS/RNAV approaches. No ILS frequency required. PFD softkey must be GPS or NAV1. Vertical guidance is managed manually.
Phase 1 — Key differences from ILS
!
RNAV — no NAV frequency required
RNAV approaches use the FMS GPS position — not a ground-based navaid. Do NOT tune a frequency for RNAV. The FMS FPLN drives lateral guidance.
What you DO need
  • RNAV approach loaded in FMS FPLN
  • PFD nav mode softkey set to GPS
  • Autopilot NAV HOLD — not APR mode
  • MDA/DA entered on APPR PAGE
Do not press APR for RNAV. APR looks for a localizer signal only. With no ILS tuned it captures nothing.
1
PFD — set nav mode to GPS
PFD lower softkeys
VSD | ADF | ILS | NAV1 | [GPS] ↑ press this
AP ALT HDG NAV ATH APR GPS TRACK Active WPT: MUCHO
GPS mode shows the FMS track deviation needle (magenta). NAV HOLD tracks this needle along the loaded flight plan.
Phase 2 — FMS FPLN loading
1
Load RNAV approach — same FPLN flow as ILS
  • Press FPLN → destination KDYS → left key
  • Select RWY 34L
  • Select approach type: RNAV 34L or GPS 34L
  • Press INSERT to activate
  • Confirm IAF, IF, FAF, and RWY34L all appear in FPLN
2
DIR — fly direct to any fix
DIRECT TO FPLN WPTS →IAF fix →IF fix →FAF fix <-- select this →RWY34L
  • Press DIR shortcut key on FMS
  • Select desired fix with left key
  • Press Direct Now bottom right key
  • Use when ATC vectors you direct to final

No glideslope for LNAV. Manually manage descent rate. Rule of thumb: groundspeed × 5 = fpm for 3° descent (e.g. 160 kt → 800 fpm).
Phase 4 — Final approach and MDA
1
LNAV MDA procedure
  • Descend to MDA — do not go below without runway in sight
  • At MDA: re-engage ALT HOLD to prevent further descent
  • Continue inbound at MDA looking for runway environment
  • Runway in sight before MAP: land
  • At MAP with no visual: immediate go-around
MDA is a hard floor. Do not descend below until over the runway threshold with landing assured.
2
Common student errors — RNAV
  • Wrong AP mode — pressing APR instead of NAV
  • MDA bust — forgetting ALT HOLD at MDA
  • Late descent — not starting at FAF; too high final
  • Wrong PFD mode — ILS selected, no GPS track shown

Autopilot Panel — KwikFlight B-1B
Autopilot Quick Reference
All autopilot modes, selectors, and PFD annunciations. Panel located left side of main instrument panel (item 1 in cockpit layout). 4 selector knobs for CRS, HDG, ALT, SPD on lower center panel (item 5).
▼

MASTER ALT HOLD HDG HOLD NAV HOLD AUTO THROT APPROACH
AP
AP — Autopilot Master
  • Toggles autopilot master on/off
  • Must be ON for any other mode to function
  • Green emissive + PFD annunciation when active
  • Disengage before touchdown or go-around
HDG
HDG HOLD — Heading Hold
  • Holds heading set on HDG selector knob
  • Rotate knob to change target heading
  • Use for ATC vectors, pattern work, departure
  • Drops out automatically when LOC captured (ILS)
NAV
NAV HOLD — Navigation Hold
  • Tracks VOR course needle (NAV1) or FMS GPS track (GPS)
  • Check PFD nav mode softkey to confirm source
  • Use for RNAV approaches, VOR approaches, en route LNAV
  • Do NOT use APR mode as substitute
Check PFD nav mode softkey — NAV HOLD behavior changes between GPS and NAV1 mode.
ALT
ALT HOLD — Altitude Hold
  • Holds altitude at moment of engagement
  • Set target on ALT selector knob before level-off
  • PFD shows ALT when active, G/S when glideslope captured
APR
APR — Approach Mode (ILS only)
  • Arms and captures ILS localizer and glideslope
  • APR box: white=off, orange=LOC armed, green=LOC captured
  • ILS frequency must be tuned in NAV page first
  • PFD nav mode must be set to ILS
APR only works with an ILS signal. Use NAV HOLD for RNAV and VOR.
ATH
AUTO THROT — Autothrottle
  • Holds airspeed set at moment of engagement
  • Rotate SPD selector knob to change target speed
  • Only operative above 2,500 ft AGL
  • Set VREF before final — manually manage throttle below 2,500 ft
A/T disengages below 2,500 ft AGL. Manually manage throttle on short final.
